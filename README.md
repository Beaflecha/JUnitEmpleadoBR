## JUnitEmpleadoBR

## 📘 Descripción

Este proyecto incluye:

- Cálculo del salario **bruto** según tipo de empleado, ventas y horas extra.
- Cálculo del salario **neto** según el tramo del salario bruto.
- Validación mediante **tests JUnit 5**.
- Manejo de casos límite y valores inválidos.

---

### 🛠️ Tecnologías utilizadas
- Java 17
- Maven
- JUnit 5
- Git y GitHub
